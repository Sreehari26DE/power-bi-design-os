A professional, personal web application built to streamline the Power BI development workflow. Generate themes, brainstorm dashboard blueprints, and draft wireframes in seconds.

## ğŸš€ Features

### 1. Theme Studio
- **JSON Generation**: Create valid Power BI theme JSON files.
- **Live Preview**: See your theme applied to a mock dashboard in real-time.
- **Accessibility First**: WCAG contrast checks and colorblind-safe palette suggestions.
- **Image Extraction**: Upload a logo to extract a brand-aligned color palette.

### 2. Idea Studio
- **Domain Blueprints**: Domain-specific recommendations for Finance, Sales, HR, and more.
- **KPI Definitions**: Structured KPI lists with definitions and pseudo-DAX formulas.
- **Data Modeling**: Suggested star schema (Facts & Dimensions) for your use case.

### 3. Wireframe Studio
- **NLP to Layout**: Describe your dashboard in natural language and get a layout.
- **Editable Canvas**: Drag and drop components to refine your design.
- **Build Guides**: Export wireframes as a reference for Power BI Desktop.

### 4. Personal Library
- **Local Persistence**: All your projects are saved locally in your browser.
- **Search & Filter**: Organize your design assets by domain or date.
